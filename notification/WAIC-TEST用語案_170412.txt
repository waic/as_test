# テスト手順や期待される結果に用いられる用語

要素
: W3Cによって定義されている、ユーザーエージェントに認識させることが可能なHTMLエレメント、及びHTMLエレメントと並列して記述することが可能なテキストの総称。

余分な情報
: ウェブサイトの利用において、ユーザーが必要としないことが自明と考えられる情報。
: 例：IMG要素のalt属性値に、画像のファイル名を設定している。

読み上げ
: 支援技術が対象の要素を分析し、分析結果をユーザーにフィードバックすること。
: 例：リンク要素の読み上げ = リンク要素に設定されているテキストを音声合成で発声する。また、ハイパーリンクであることを意味する音声や効果音を再生する。

読み上げさせる
: 読み上げを行うために、対象の要素上に読み上げカーソルを移動する操作。

何らかの手段
: ユーザーエージェントおよび支援技術に搭載されている機能の内、下記の要件の全てに当てはまる機能。

  1. 提供者によるマニュアル上で規定されている
  2. JIS X 6002-1980に規定されているキー配列の組み合わせにより実行できる
     例えば、「F13(124)」や、「戻る(166)」などのキー押下が必要な機能、及びタッチパッドなどのボタン押下が必要な機能は、この要件を満たさない。
  3. 他のユーザーインタフェースの操作を代行する機能ではない
     例えば、マウスの右クリックを行う機能や、OCRを実行する機能は、この要件を満たさない。

差し掛かった時
: 支援技術の利用中に、別の要素の位置から当該要素の位置へ読み上げカーソルが移動した時

## 不要になった用語

### 通知される
元の定義
: スクリーンリーダーが当該要素に差し掛かった時に、必要な情報を自動的に伝えること。
  例：　見出しに差し掛かった時に、それが見出しであることや見出しレベルなどを自動的に伝える

削除理由
: - 「通知される」という動詞に、名詞や関係代名詞などが混在しているため混乱を招く。
  - 現状のテストでは、何らかのユーザーの操作に対してスクリーンリーダーが行うフィードバックを指しているので、「XX要素に差しかかった時、YYが通知される」という書き方が自然に伝わる。


# スクリーンリーダーのモードや利用頻度の高いコマンドについて

読み上げカーソル
: スクリーンリーダーが読み上げを行う要素を示すカーソル
: PC-Talker「仮想カーソル」、NVDA「(ブラウズモードでの)キャレット」、JAWS「仮想PCカーソル」

テキストモード
: ウェブサイトを閲覧するために用いる操作モード。矢印キーやジャンプ機能などを用いて、ウェブページを読み上げさせることができる。
このモードの利用中は、矢印キーや文字キーなどのキーボード操作が読み上げカーソルの移動のために用いられる。このため、一般的なユーザーエージェントが提供している機能のほとんど(例「矢印キーによるスクロール操作」、「フォームなどへの情報の入力」)が制限される。
: PC-Talker/NetReader「行読みモード」「ナビゲーションキーモード」、NVDA
「ブラウズモード」、JAWS「仮想PCカーソルモード」

フォームモード
: ウェブサイトのフォームなどへ情報を入力する時に使用するモード。テキストモード利用中とは対照的に、利用者のキー操作がユーザーエージェントにバイパスされるので、入力欄への文字入力や、リストボックスからの項目選択などの操作が可能となる。一方で、要素間を移動するためには、ユーザーエージェントにより提供されている標準の操作(タブキーなど)しか利用できなくなる。
: PC-Talker「エディットモード」、NVDA「フォーカスモード」、JAWS「フォーム
モード」

テーブルモード
: ウェブサイトのテーブル要素内で使用できるモード。テーブルモード使用中は、矢印キーを押下した方向に隣接するセルを読み上げさせることができる。また、テーブル要素の外側にある要素への読み上げカーソルの移動が無効となる。
: NetReader「テーブルモード」、NVDA/JAWS「テーブル内での操作(テキストモードの1操作として定義されている)」

ページプロパティ
: ウェブページの全体像を確認するための機能。以下のような情報の閲覧のみが可能となる。

  - ウェブページのタイトル
  - ウェブページのURL
  - フレームが利用されている場合、存在するフレーム数
  - ウェブページ内に存在する見出しやリンク、テーブルなどの要素の数

ジャンプ機能
: テキストモード利用中に、任意の位置にある要素を検索し、該当した要素の位置へ読み上げカーソルを移動する機能。
: PC-Talker/NVDA「1文字ナビゲーション」、NetReader「クィックビュー」、JAWS「ナビゲーション クイック キー」

読み飛ばす
: ジャンプ機能を利用して、移動前の位置から移動後の位置の手前までに存在する全ての要素を読み上げすることなく、読み上げカーソルを移動する行為。
