# テスト手順 (音声閲覧環境)

# 期待される結果 (音声閲覧環境)

直前または直後の nav 要素に直接移動するための何らかの手段が提供されている

# テスト実施時の注意点 (音声閲覧環境)

あくまで nav 要素を目標とした移動（ジャンプ）であること

- Tab キーによるフォーカス移動などによる nav 要素内の a 要素に対する直接移動は、nav 要素に対する直接移動とは見なさない
- マークアップ順に従って h1 要素や p 要素の次に nav 要素に移動することは、nav 要素に対する直接移動とは見なさない
