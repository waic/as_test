# テスト ID

WAIC-TEST-0010-08

# テストのタイトル

データセルと見出しセル (id 属性及び headers 属性)

# テストの目的

id 属性及び headers 属性を指定した場合、テーブルのデータセルと見出しセル (列見出し) が関連付いているかの確認

# テストの対象となる達成基準 (複数)

1.3.1

# 関連する達成方法 (複数)

# テストコード (テストファイルへのリンク)

