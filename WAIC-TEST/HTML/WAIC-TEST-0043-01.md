WAIC-TEST-0043-01

# テストのタイトル
リージョンとランドマークに名前 (name) を付けるために、aria-labelledby を使用する

# テストの目的

complementary ランドマークに対して aria-labelledby を使用して名前 (name) を与えた場合に、ランドマークと名前 (name) が認識され、通知されるかどうかを確認する。
